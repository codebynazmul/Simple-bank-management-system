#  Bank Management System in C

A simple, class-wise **Bank Management System** built in pure **C language** â€”  
designed to manage customer accounts, deposits, withdrawals, and balance checks.  
All data is stored locally using file handling.

---

##  Features

 * Add new account  
 * Deposit and withdraw money  
 * Check account balance  
 * Delete an account  
 * Data stored permanently in a file (`bank_data.txt`)  
 * Each section loads on a fresh console screen  
 * Closes with a signature screen showing developer credit

---

##  Important Note

 **This program will not run properly on any online compiler**  
because it uses **local file handling (read/write operations)** to save data.  

To experience the full functionality, please **run it on your local computer** using:
- GCC Compiler  
- Code::Blocks  
- Dev-C++  
- or any offline C environment

---

##  Technologies Used

- **Language:** C  
- **Compiler:** GCC / Code::Blocks / Turbo C  
- **Data Storage:** File Handling (`bank_data.txt`)  
- **Platform:** Cross-platform (Windows / Linux)
